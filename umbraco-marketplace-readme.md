# Conditional Displayers

Umbraco package that offers a collection of editors that allow to hide or display other properties in the same Document Type.

This package offers a checkbox, a dropdown and a radio list datatype editors that work like the core ones (in fact they are based in the same code), but allows to hide or display other properties in the same Document Type.

## Installation

### Nuget
![NuGet Downloads](https://img.shields.io/nuget/dt/Our.Umbraco.ConditionalDisplayers?label=Our.Umbraco.ConditionalDisplayers)

- **Less than Umbraco 16**: Use Conditional Displayers version 3
